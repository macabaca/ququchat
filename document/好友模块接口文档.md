# 好友模块接口文档 (UserHandler)

本文档详细描述了 `internal/api/handler/user_handler.go` 中的接口逻辑，涵盖所有成功和失败的返回情况。

**Base URL**: `/api`

---

## 1. 发送好友请求 (AddFriend)

- **URL**: `/friends/add`
- **Method**: `POST`
- **认证**: 需要 (Header: `Authorization: Bearer <token>`)

### 请求参数
**Body (JSON)**:
```json
{
    "target_user_code": 10001,  // (必填) int64, 目标用户的 user_code
    "message": "我是..."        // (选填) string, 验证消息
}
```

### 返回情况汇总

| 场景 | HTTP状态码 | 返回内容 (JSON) | 说明 |
| :--- | :--- | :--- | :--- |
| **未登录** | 401 Unauthorized | `{"error": "未登录"}` | Header 中缺少 Token 或 Token 无效 |
| **参数错误** | 400 Bad Request | `{"error": "请求体格式错误或目标用户无效"}` | JSON 解析失败或 `target_user_code <= 0` |
| **用户不存在** | 404 Not Found | `{"error": "用户不存在"}` | 数据库中找不到该 `user_code` 对应的用户 |
| **查询用户异常** | 500 Internal Server Error | `{"error": "查询用户失败"}` | 数据库查询目标用户时发生未知错误 |
| **添加自己** | 400 Bad Request | `{"error": "不能添加自己为好友"}` | 目标用户 ID 与当前用户 ID 相同 |
| **已经是好友** | 200 OK | `{"message": "已是好友", "friend": {...}}` | 双方已经是好友关系，直接返回好友信息 |
| **查询好友关系异常** | 500 Internal Server Error | `{"error": "查询好友关系失败"}` | 检查好友关系时数据库报错 (非 RecordNotFound) |
| **已有待处理请求** | 200 OK | `{"message": "已存在待处理的好友请求", "request": {...}}` | 双方之间已存在状态为 `pending` 的请求 |
| **查询请求异常** | 500 Internal Server Error | `{"error": "查询已有好友请求失败"}` | 检查现有请求时数据库报错 (非 RecordNotFound) |
| **并发重复请求** | 200 OK | `{"message": "已存在待处理的好友请求", "request": {...}}` | 创建请求失败后，二次检查发现请求已存在（并发场景） |
| **创建请求异常** | 500 Internal Server Error | `{"error": "创建好友请求失败"}` | 写入新请求到数据库时失败 |
| **发送成功** | 201 Created | `{"message": "好友请求已发送", "request": {...}}` | 请求成功创建 |

## 5. 响应好友请求 (RespondFriendRequest)

- **URL**: `/friends/requests/respond`
- **Method**: `POST`
- **认证**: 需要

### 请求参数
**Body (JSON)**:
```json
{
    "request_id": "uuid-string", // (必填) 好友请求的 ID
    "action": "accept"           // (必填) "accept" (接受) 或 "reject" (拒绝)
}
```

### 返回情况汇总

| 场景 | HTTP状态码 | 返回内容 (JSON) | 说明 |
| :--- | :--- | :--- | :--- |
| **未登录** | 401 Unauthorized | `{"error": "未登录"}` | Header 中缺少 Token 或 Token 无效 |
| **参数错误** | 400 Bad Request | `{"error": "请求体格式错误或操作无效"}` | JSON 解析失败、ID为空或 action 不是 accept/reject |
| **请求不存在** | 404 Not Found | `{"error": "好友请求不存在"}` | 数据库未找到指定 ID 的请求 |
| **查询请求异常** | 500 Internal Server Error | `{"error": "查询好友请求失败"}` | 查询请求时数据库报错 (非 RecordNotFound) |
| **无权处理** | 403 Forbidden | `{"error": "无权处理该好友请求"}` | 请求的目标用户 (ToUserID) 不是当前用户 |
| **状态已过期** | 400 Bad Request | `{"error": "该好友请求已被处理"}` | 请求状态不是 `pending` |
| **拒绝-更新异常** | 500 Internal Server Error | `{"error": "更新好友请求状态失败"}` | 执行拒绝操作更新数据库时失败 |
| **拒绝成功** | 200 OK | `{"message": "已拒绝好友请求", "request": {...}}` | 成功拒绝 |
| **接受-事务异常** | 500 Internal Server Error | `{"error": "接受好友请求失败"}` | 接受操作（更新状态+创建好友关系）事务执行失败 |
| **接受成功** | 200 OK | `{"message": "已接受好友请求", "request": {...}}` | 成功接受并建立好友关系 |
