# 群组模块接口文档

本文档总结了 `d:\study\project\ququchat\internal\api\handler\group_handler.go` 中定义的群组相关接口。所有接口均需要 `Authorization` Header 携带 JWT Token。

## 1. 创建群组

- **URL**: `/api/groups/create`
- **Method**: `POST`
- **功能**: 创建一个新的群聊。
- **请求体 (JSON)**:
  ```json
  {
    "name": "我的群组",       // 必填，群名称
    "member_ids": ["1001", "1002"] // 选填，初始邀请的成员ID列表
  }
  ```
- **成功响应 (201 Created)**:
  ```json
  {
    "group": {
      "id": "uuid-string",
      "name": "我的群组",
      "owner_id": "current-user-id",
      "member_count": 3,
      "created_at": "2023-10-27T10:00:00Z"
    }
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}`
  - `400 Bad Request`: `{"error": "请求体格式错误"}`
  - `400 Bad Request`: `{"error": "群名称不能为空"}`
  - `500 Internal Server Error`: `{"error": "创建群失败"}`
  - `500 Internal Server Error`: `{"error": "创建群成员失败"}`

## 2. 获取群组详情

- **URL**: `/api/groups/:group_id`
- **Method**: `GET`
- **功能**: 获取指定群组的详细信息。
- **路径参数**:
  - `group_id`: 群组的唯一标识 ID。
- **成功响应 (200 OK)**:
  ```json
  {
    "group": {
      "id": "uuid-string",
      "name": "群名称",
      "owner_id": "owner-id",
      "member_count": 10,
      "my_role": "owner", // owner, admin, member
      "created_at": "...",
      "updated_at": "..."
    }
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}`
  - `400 Bad Request`: `{"error": "缺少群ID"}`
  - `404 Not Found`: `{"error": "群不存在"}`
  - `500 Internal Server Error`: `{"error": "查询群信息失败"}`
  - `403 Forbidden`: `{"error": "未加入该群"}` (尝试查询非自己加入的群)
  - `500 Internal Server Error`: `{"error": "查询群成员失败"}`

## 3. 获取我的群组列表

- **URL**: `/api/groups/my`
- **Method**: `GET`
- **功能**: 获取当前用户加入的所有群组。
- **成功响应 (200 OK)**:
  ```json
  {
    "groups": [
      {
        "id": "...",
        "name": "...",
        "owner_id": "...",
        "member_count": 5,
        "my_role": "member",
        "created_at": "..."
      }
    ]
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}`
  - `500 Internal Server Error`: `{"error": "查询群成员关系失败"}`
  - `500 Internal Server Error`: `{"error": "查询群信息失败"}`

## 4. 解散群组

- **URL**: `/api/groups/:group_id/dismiss`
- **Method**: `POST`
- **功能**: 解散群组（仅群主可用）。
- **路径参数**:
  - `group_id`: 群组 ID。
- **成功响应 (200 OK)**:
  ```json
  {
    "message": "群已解散"
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}`
  - `400 Bad Request`: `{"error": "缺少群ID"}`
  - `404 Not Found`: `{"error": "群不存在"}`
  - `500 Internal Server Error`: `{"error": "查询群信息失败"}`
  - `403 Forbidden`: `{"error": "只有群主可以解散群"}`
  - `500 Internal Server Error`: `{"error": "解散群失败"}`

## 5. 邀请成员加入

- **URL**: `/api/groups/:group_id/members/add`
- **Method**: `POST`
- **功能**: 邀请用户加入群组（仅群主或管理员）。
- **路径参数**:
  - `group_id`: 群组 ID。
- **请求体 (JSON)**:
  ```json
  {
    "user_ids": ["uid1", "uid2"] // 必填，要添加的用户ID列表
  }
  ```
- **成功响应 (200 OK)**:
  ```json
  {
    "message": "操作完成",
    "added_count": 2
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}`
  - `400 Bad Request`: `{"error": "缺少群ID"}`
  - `400 Bad Request`: `{"error": "请求参数错误"}`
  - `400 Bad Request`: `{"error": "未指定要添加的用户"}`
  - `404 Not Found`: `{"error": "群不存在"}`
  - `500 Internal Server Error`: `{"error": "查询群信息失败"}`
  - `403 Forbidden`: `{"error": "您不是群成员"}`
  - `403 Forbidden`: `{"error": "权限不足，仅群主或管理员可拉人"}`

## 6. 移除群成员

- **URL**: `/api/groups/:group_id/members/remove`
- **Method**: `POST`
- **功能**: 移除群成员（群主可踢任何人，管理员可踢普通成员）。
- **路径参数**:
  - `group_id`: 群组 ID。
- **请求体 (JSON)**:
  ```json
  {
    "user_id": "target-uid" // 必填，要移除的用户ID
  }
  ```
- **成功响应 (200 OK)**:
  ```json
  {
    "message": "成员已移除"
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}`
  - `400 Bad Request`: `{"error": "缺少群ID"}`
  - `400 Bad Request`: `{"error": "请求参数错误"}`
  - `403 Forbidden`: `{"error": "您不是群成员"}`
  - `404 Not Found`: `{"error": "目标用户不在群内"}`
  - `403 Forbidden`: `{"error": "权限不足"}`
  - `500 Internal Server Error`: `{"error": "移除成员失败"}`

## 7. 退出群组

- **URL**: `/api/groups/:group_id/leave`
- **Method**: `POST`
- **功能**: 主动退出群组（群主不可直接退出，需先转让或解散）。
- **路径参数**:
  - `group_id`: 群组 ID。
- **成功响应 (200 OK)**:
  ```json
  {
    "message": "已退出群"
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}`
  - `400 Bad Request`: `{"error": "缺少群ID"}`
  - `400 Bad Request`: `{"error": "您不在该群中"}`
  - `500 Internal Server Error`: `{"error": "查询成员状态失败"}`
  - `403 Forbidden`: `{"error": "群主不能直接退出，请先转让群主或解散群"}`
  - `500 Internal Server Error`: `{"error": "退出群失败"}`

## 8. 获取群成员列表

- **URL**: `/api/groups/:group_id/members`
- **Method**: `GET`
- **功能**: 获取群组内的所有成员列表。
- **路径参数**:
  - `group_id`: 群组 ID。
- **成功响应 (200 OK)**:
  ```json
  {
    "members": [
      {
        "user_id": "...",
        "username": "...",
        "nickname": "...", // 群内昵称或用户昵称
        "role": "owner",   // owner, admin, member
        "joined_at": "..."
      }
    ]
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}`
  - `400 Bad Request`: `{"error": "缺少群ID"}`
  - `403 Forbidden`: `{"error": "您不是该群成员"}`
  - `500 Internal Server Error`: `{"error": "查询群成员失败"}`
