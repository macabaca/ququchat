# 群组模块接口文档

本文档总结了 `d:\study\project\ququchat\internal\api\handler\group_handler.go` 中定义的群组相关接口。所有接口均需要 `Authorization` Header 携带 JWT Token。

## 1. 创建群组

- **URL**: `/api/groups/create`
- **Method**: `POST`
- **功能**: 创建一个新的群聊。
- **请求体 (JSON)**:
  ```json
  {
    "name": "我的群组",       // 必填，群名称
    "member_ids": ["1001", "1002"] // 选填，初始邀请的成员ID列表
  }
  ```
- **成功响应 (201 Created)**:
  ```json
  {
    "group": {
      "id": "uuid-string",
      "name": "我的群组",
      "owner_id": "current-user-id",
      "member_count": 3,
      "created_at": "2023-10-27T10:00:00Z"
    }
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}` - Header 中缺少 Authorization Token 或 Token 无效。
  - `400 Bad Request`: `{"error": "请求体格式错误"}` - 请求体 JSON 格式不正确。
  - `400 Bad Request`: `{"error": "群名称不能为空"}` - 请求体中 `name` 字段为空。
  - `500 Internal Server Error`: `{"error": "创建群失败"}` - 数据库错误，无法创建群记录。
  - `500 Internal Server Error`: `{"error": "创建群成员失败"}` - 数据库错误，无法创建群主或初始成员记录。

## 2. 获取群组详情

- **URL**: `/api/groups/:group_id`
- **Method**: `GET`
- **功能**: 获取指定群组的详细信息。
- **路径参数**:
  - `group_id`: 群组的唯一标识 ID。
- **成功响应 (200 OK)**:
  ```json
  {
    "group": {
      "id": "uuid-string",
      "name": "群名称",
      "owner_id": "owner-id",
      "member_count": 10,
      "my_role": "owner", // owner, admin, member
      "created_at": "...",
      "updated_at": "..."
    }
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}` - Header 中缺少 Authorization Token 或 Token 无效。
  - `400 Bad Request`: `{"error": "缺少群ID"}` - URL 中未提供 `group_id`。
  - `404 Not Found`: `{"error": "群不存在"}` - 指定 ID 的群组不存在或非群组类型。
  - `500 Internal Server Error`: `{"error": "查询群信息失败"}` - 数据库查询群信息时发生未知错误。
  - `403 Forbidden`: `{"error": "未加入该群"}` - 当前用户不是该群成员（或已退群）。
  - `500 Internal Server Error`: `{"error": "查询群成员失败"}` - 数据库查询用户成员关系时发生未知错误。

## 9. 批量设置管理员

- **URL**: `/api/groups/:group_id/admins/add`
- **Method**: `POST`
- **功能**: 批量将群成员提升为管理员（仅群主可用）。
- **路径参数**:
  - `group_id`: 群组 ID。
- **请求体 (JSON)**:
  ```json
  {
    "user_ids": ["uid1", "uid2"] // 必填，要设置为管理员的用户ID列表
  }
  ```
- **成功响应 (200 OK)**:
  ```json
  {
    "message": "操作成功",
    "updated_count": 2
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}` - Header 中缺少 Authorization Token 或 Token 无效。
  - `400 Bad Request`: `{"error": "缺少群ID"}` - URL 中未提供 `group_id`。
  - `400 Bad Request`: `{"error": "请求参数错误"}` - 请求体 JSON 格式不正确。
  - `400 Bad Request`: `{"error": "未指定用户"}` - 请求体中 `user_ids` 列表为空。
  - `404 Not Found`: `{"error": "群不存在"}` - 指定 ID 的群组不存在或非群组类型。
  - `403 Forbidden`: `{"error": "只有群主可以设置管理员"}` - 操作者不是群主。
  - `500 Internal Server Error`: `{"error": "设置管理员失败"}` - 数据库更新成员角色时发生错误。

## 3. 获取我的群组列表

- **URL**: `/api/groups/my`
- **Method**: `GET`
- **功能**: 获取当前用户加入过的所有群组（包含正常在群、已退群、已解散的群组）。
- **成功响应 (200 OK)**:
  ```json
  {
    "groups": [
      {
        "id": "...",
        "name": "...",
        "owner_id": "...",
        "member_count": 5,
        "my_role": "member",
        "created_at": "...",
        "status": "active" // active(正常), left(已退群), dismissed(已解散)
      }
    ]
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}` - Header 中缺少 Authorization Token 或 Token 无效。
  - `500 Internal Server Error`: `{"error": "查询群成员关系失败"}` - 数据库查询用户加入的群列表时发生错误。
  - `500 Internal Server Error`: `{"error": "查询群信息失败"}` - 数据库查询群详细信息时发生错误。

## 5. 邀请成员加入

- **功能**: 邀请用户加入群组（仅群主或管理员）。
- **路径参数**:
  - `group_id`: 群组 ID。
- **请求体 (JSON)**:
  ```json
  {
    "user_ids": ["uid1", "uid2"] // 必填，要添加的用户ID列表
  }
  ```
- **成功响应 (200 OK)**:
  ```json
  {
    "message": "操作完成",
    "added_count": 2
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}` - Header 中缺少 Authorization Token 或 Token 无效。
  - `400 Bad Request`: `{"error": "缺少群ID"}` - URL 中未提供 `group_id`。
  - `400 Bad Request`: `{"error": "请求参数错误"}` - 请求体 JSON 格式不正确。
  - `400 Bad Request`: `{"error": "未指定要添加的用户"}` - 请求体中 `user_ids` 列表为空。
  - `404 Not Found`: `{"error": "群不存在"}` - 指定 ID 的群组不存在或非群组类型。
  - `500 Internal Server Error`: `{"error": "查询群信息失败"}` - 数据库查询群信息时发生未知错误。
  - `403 Forbidden`: `{"error": "您不是群成员"}` - 操作者不是该群成员。
  - `403 Forbidden`: `{"error": "权限不足，仅群主或管理员可拉人"}` - 操作者是普通成员，无权邀请他人（需群主或管理员）。

## 6. 移除群成员

- **URL**: `/api/groups/:group_id/members/remove`
- **Method**: `POST`
- **功能**: 移除群成员（群主可踢任何人，管理员可踢普通成员）。
- **路径参数**:
  - `group_id`: 群组 ID。
- **请求体 (JSON)**:
  ```json
  {
    "user_id": "target-uid" // 必填，要移除的用户ID
  }
  ```
- **成功响应 (200 OK)**:
  ```json
  {
    "message": "成员已移除"
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}` - Header 中缺少 Authorization Token 或 Token 无效。
  - `400 Bad Request`: `{"error": "缺少群ID"}` - URL 中未提供 `group_id`。
  - `400 Bad Request`: `{"error": "请求参数错误"}` - 请求体 JSON 格式不正确。
  - `403 Forbidden`: `{"error": "您不是群成员"}` - 操作者不是该群成员。
  - `404 Not Found`: `{"error": "目标用户不在群内"}` - 试图移除的用户不在该群中。
  - `403 Forbidden`: `{"error": "权限不足"}` - 普通成员尝试踢人，或管理员尝试踢管理员/群主。
  - `500 Internal Server Error`: `{"error": "移除成员失败"}` - 数据库更新成员状态为已离开时发生错误。

## 7. 退出群组

- **URL**: `/api/groups/:group_id/leave`
- **Method**: `POST`
- **功能**: 主动退出群组（群主不可直接退出，需先转让或解散）。
- **路径参数**:
  - `group_id`: 群组 ID。
- **成功响应 (200 OK)**:
  ```json
  {
    "message": "已退出群"
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}` - Header 中缺少 Authorization Token 或 Token 无效。
  - `400 Bad Request`: `{"error": "缺少群ID"}` - URL 中未提供 `group_id`。
  - `400 Bad Request`: `{"error": "您不在该群中"}` - 当前用户未加入该群或已退群。
  - `500 Internal Server Error`: `{"error": "查询成员状态失败"}` - 数据库查询成员信息时发生未知错误。
  - `403 Forbidden`: `{"error": "群主不能直接退出，请先转让群主或解散群"}` - 群主尝试退出群组（需先解散或转让）。
  - `500 Internal Server Error`: `{"error": "退出群失败"}` - 数据库更新成员状态为已离开时发生错误。

## 8. 获取群成员列表

- **URL**: `/api/groups/:group_id/members`
- **Method**: `GET`
- **功能**: 获取群组内的所有成员列表。
- **路径参数**:
  - `group_id`: 群组 ID。
- **成功响应 (200 OK)**:
  ```json
  {
    "members": [
      {
        "user_id": "...",
        "username": "...",
        "nickname": "...", // 群内昵称或用户昵称
        "avatar_attachment_id": "attachment-uuid",
        "role": "owner",   // owner, admin, member
        "joined_at": "..."
      }
    ]
  }
  ```
- **错误响应**:
  - `401 Unauthorized`: `{"error": "未登录"}` - Header 中缺少 Authorization Token 或 Token 无效。
  - `400 Bad Request`: `{"error": "缺少群ID"}` - URL 中未提供 `group_id`。
  - `403 Forbidden`: `{"error": "您不是该群成员"}` - 当前用户不是该群成员（无法查看成员列表）。
  - `500 Internal Server Error`: `{"error": "查询群成员失败"}` - 数据库查询成员列表时发生错误。
